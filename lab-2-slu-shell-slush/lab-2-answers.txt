	the built-in command cd dir

	EOF exit using Ctrl+D

	Each input line is limited to 256 characters.
	The shell does not support quoted strings or advanced shell syntax.
	The exact exec error message depends on perror(), so it may say "No such file or directory" instead of exactly "Not found."
